# 创建第一个项目

本教程将带你从新建到保存，完成一个简易的首个电机音项目。

完成后，你将：

## 第 1 步：创建项目

1. 打开首页。
2. 选择 **新建项目 New Project**。
3. 输入项目名称。
4. 设置合理的最高速度。
5. 设置加速度与制动默认值。
6. 进入编辑器。

## 第 2 步：确认激活轨道

如果编辑器已存在可用默认轨道，可直接使用。否则：

1. 打开轨道层面板。
2. 新增一条轨道。
3. 激活它。
4. 改成有语义的名称。

## 第 3 步：分配音频

1. 在轨道详情中打开文件选择器。
2. 选择 `wav` 或 `ogg` 文件。
3. 确认激活轨道详情里显示了该文件名。

## 第 4 步：添加初始曲线形状

使用图表编辑器或列表编辑器创建一个简单起步曲线。先不追求复杂结果。实用起步方式是：

- 在接近零速处放一个点
- 在中速区放一个点
- 在高速度区附近放一个点

若你希望立刻听到明显变化，建议音高与音量都先各做一轮基础形状。

## 第 5 步：试听

1. 将模拟器模式切到 **牵引 Traction**。
2. 设定一个当前速度。
3. 按 `Space`。
4. 判断轨道行为是否大致符合预期。

如果没有声音，优先检查：

- 轨道是否已分配文件
- 轨道是否被静音
- 轨道是否被隐藏

## 第 6 步：保存项目

1. 按 `Ctrl+S` 或使用保存命令。
2. 必要时选择保存位置和文件名。
3. 确认项目已写入为 `.msep`。

## 结果

你现在已经拥有一个可重开、可继续打磨的真实项目。
