# 列表编辑器

工作区的可视编辑器是主要创作界面，但不是唯一入口。Motor Sound Editor 还提供 **列表视图编辑器 (List View Editor)**，用于基于表格的关键帧处理。

对大量关键帧进行批量操作的场景，列表视图提供了比图形化曲线更直接的数据反馈。

![列表视图](/docs/listview.png)

---

## 1. 核心功能

列表编辑器将曲线上的每一个关键帧映射为表格中的一行。通过这种形式，你可以像操作 Excel 电子表格一样管理声音数据。

* **精确数值输入**：在图表区手动拖拽关键帧可能难以达到理想的整数值。在列表视图中，你可以直接输入具体的数值（如速度为 `60.5 km/h`，音量为 `0.85`）。
* **全局总览**：在一个视图内查看当前轨道中所有的关键帧分布，快速发现是否存在逻辑上的突跳或冗余点。
* **多模式支持**：列表视图会根据编辑器当前选定的**工况模式（牵引 Traction / 制动 Brake）**以及**曲线类型（音高 Pitch / 音量 Volume）**动态刷新数据。

---

## 2. 界面组成


表格主要包含以下几列：

1. **位置 (X - Speed)**：横轴坐标。
2. **数值 (Y - Value)**：纵轴坐标。代表在该节点下的音高 (Pitch)倍率 或响度百分比(Volume)。

---

## 3. 关键特性

### 双向同步 

列表编辑器与主图表工作区是完全同步的。

- 当你在列表中修改数值后，点击应用，主工作区的曲线形状会发生形变。
- 反之，在图表区移动关键帧，列表中的数值也会动态该变。

### 排序逻辑

列表视图具有自动排序功能。关键帧会始终根据其 **X 轴坐标（位置）** 从小到大排列。这意味着当你修改某个点的 X 值并导致其超过相邻点时，该行在列表中的顺序会自动重排，以确保逻辑顺序的一致性。

---

## 4. 右键上下文菜单

为了进一步提升操作效率，无论是在图表工作区的曲线上，还是在列表编辑器的表格行中，你都可以通过 **鼠标右键单击** 唤出上下文菜单，快速执行针对关键帧和数据的进阶操作。

![右键上下文菜单](/docs/listcontexmenu.png)


右键菜单聚合了最常用的编辑命令，使你无需将鼠标频繁移动到顶部工具栏。典型的右键菜单功能包括：


* **数值与关键帧操作**：
    * **同步牵引(制动)到制动(牵引)**：将牵引(制动)数据同步至制动。
    * **添加关键帧**：快速插入一个新的关键帧点。自动排序。
    * **删除**：快速移除不再需要的关键帧或列表行。
