# 快速开始

Motor Sound Editor 面向希望在可视化编辑器中创作分层电机音效的用户，而不是手工维护原始模拟器表格。当前版本重点覆盖桌面应用里已经落地的三类核心流程：项目创作、实时预览与导出打包。

![Motor Sound Editor 首页](/homepage.png)

## 1. 准备工作

### 操作系统要求

**Windows** 是目前软件的主要支持平台。

### 下载与安装

- **获取软件**： 你可以前往 [项目下载页面] 或 [GitHub Releases] 获取最新的 Windows 安装包。

- **快速安装**： 下载完成后，运行安装程序。无需安装额外的运行库（Tauri 已将必要组件打包）。

### 数据与素材

- **素材：** 至少一段电机音源文件（支持 `.wav` 或 `.ogg` 格式）。
- **参数：** 车辆的基本性能参数，如最高时速、加速度和减速度。

## 2. 核心工作流

你可以通过以下 6 个步骤快速体验软件的核心功能：

### 第一步：创建项目

![新建文件](/docs/newproject.png)

1. 启动应用，在主页点击 **New Project**。
2. 输入项目名称（例如：MyTrainMotor）。
3. 设定 **Max Speed**（最高速度）、**Acceleration**（加速度）等默认参数。
4. 点击确认进入编辑器。

### 第二步：管理轨道与导入音频

![新建文件](/docs/importaudio.png)

1. 在左侧 **Track Layers** 面板中，确保有一个激活的轨道。
2. 在 **Track Details** 区域点击文件选择图标。
3. 选择你的 `.wav` 或 `.ogg` 音频文件。此时你可以看到轨道名称已更新。

### 第三步：绘制音调与音量曲线

![绘制](/docs/trackline.png)

1. **工具选择：** 在工具栏选择**关键帧模式**（Keyframe）。
2. **绘制牵引力曲线：** 在图表中为 **Traction Pitch** (牵引音调) 和 **Traction Volume** (牵引音量) 添加关键点，左键为添加点，右键为删除点。
3. **精细调整：** 你可以直接在图表中切换为**移动模式**（Move Mode）拖动关键点，也可以在 **List Editor** (列表编辑器) 中手动输入数值。

### 第四步：实时预览

![实时预览](/docs/rt-preview.png)

1. 按下 `Space` 键开始播放音频。
2. 使用快捷键 `W`（增加动力/切至牵引）或 `S`（增加制动/切至制动）来模拟列车运行状态。
3. 在左侧修改当前速度（Current Speed）或切换**选择模式**（Select Mode）之后单击空白处，可以快速更改速度，以辅助监听声音的变化。

### 第五步：保存项目

* 按下 `Ctrl + S` 将项目保存为 `.msep` 格式。
* **注意：** `.msep` 是本软件的专属工程格式，它会一站式打包你的所有设置、曲线数据和音频素材，方便你随时移动或重新编辑。

### 第六步：导出 BVE 格式压缩包

![导出](/docs/output.png)

1. 点击工具栏或菜单中的 **Export** 图标。
2. 在弹出的对话框中选择 **BVE** 目标格式。
3. 选择采样率（建议 44100Hz），点击导出。
4. 软件将生成一个 `.zip` 压缩包，内含 BVE 规范的 `vehicle.txt`、`motornoise.txt` 及相关 WAV 音频（导出时会自动转换到对应标准的音频格式）。


## 3. 常用快捷键

熟练使用快捷键可以极大提高编辑效率：

| 快捷键 | 功能 |
| :--- | :--- |
| **Space** | 播放 / 暂停预览 |
| **W** | 模拟状态向上切（牵引 Traction） |
| **S** | 模拟状态向下切（制动 Brake） |
| **Ctrl + S** | 保存项目 |
| **Ctrl + Z / Y** | 撤销 / 重做 |
| **Delete** | 删除选中的关键点 |
