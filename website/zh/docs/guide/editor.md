# 编辑器工作区

编辑器工作区是 Motor Sound Editor 的核心操作环境。在这里，你可以管理多个声音轨道，分配音频资产，并通过直观的曲线图表来精细调节列车在不同牵引力 (Traction) 与制动力 (Brake) 下的音高 (Pitch) 与音量 (Volume) 变化规律。

![标注编辑页面](/docs/lablededit.png)

工作区在设计上参考了主流三维动画与非线性编辑软件（如 Blender 的曲线编辑器），采用模块化布局，主要分为以下四个关键区域：

1. **工具栏 (Toolbar)**：提供全局工具模式切换与项目操作。
2. **轨道侧栏 (Track Sidebar)**：位于左侧，用于管理音轨层级、音频文件分配与轨道状态。
3. **工作区 (Workspace)**：占据中央最大面积的曲线编辑与关键帧操作区域。
4. **底部状态栏 (Status Bar)**：位于底部，用于实时声音预览与速度/状态控制。

---

## 1. 工具栏 (Toolbar)

顶部工具栏包含了影响整个工作区交互行为的全局控制选项。

![工具](/docs/tools.png)

* **工具模式 (Tools Mode)**：决定你在主图表工作区中与关键帧或曲线交互的方式：
    * **选择工具 (Select)**：默认模式。用于点选关键帧。
    * **移动工具 (Move)**：用于自由拖拽已选中的关键帧，改变其数值。
    * **关键帧工具 (Keyframe)**：在曲线上左键单击以插入新的关键帧，或右键单击快捷删除不需要的关键帧。
* **吸附与对齐 (Snap)**：按住Shift开启。开启后，移动关键帧时会自动吸附到特定的网格值（speed为0.5的倍数，value为0.2的倍数）。

---

## 2. 轨道侧栏 (Track Sidebar)

侧栏的作用类似于混音器与图层面板的结合体，你可以在这里组织和管理所有参与合成的电机音音轨。

<img src="/docs/trackbar.png" width="200" alt="音轨栏">

* **轨道列表 (Track List)**：自上而下显示当前项目中的所有轨道。点击任意轨道即可将其设为 **活动轨道 (Active Track)**，主图表区将立即同步显示该轨道对应的所有曲线。
* **资产分配 (Asset Assignment)**：每个轨道右侧或下方会显示其绑定的本地音频文件（如 `.wav`）。你可以点击替换或导入新的源文件。
* **轨道状态控制 (Track Controls)**：
    * **可见性 (👁 显示/隐藏)**：切换该轨道的曲线在主图表中的显示状态。在处理包含十几条轨道的复杂项目时，隐藏非活动轨道可以有效保持视图整洁。
    * **静音 (Mute)**：在进行底部音频预览时，暂时屏蔽该轨道的声音。

## 3. 主图表工作区 (Workspace)

这是编辑电机声音动态变化的核心组件。在这个二维坐标系中，横轴 (X轴) 通常代表列车的运行速度或转速，纵轴 (Y轴) 代表音高倍率或音量百分比。

![工作区](/docs/labledworkspace.png)

为了模拟真实的电机反馈，每一个音频轨道通常包含两套独立的曲线环境：

- **牵引与制动 (Traction / Brake)**：列车在牵引（加速）和制动（减速）时的逆变器发声逻辑通常是不同的。

**牵引与制动的数据是分离的，当模式为牵引时会显示牵引的音轨与关键帧数据，制动同理。在软件中惰行会显示牵引的音轨与关键帧**
- **音高 (Pitch) 曲线**：控制音频在不同速度下的播放速率或频率偏移。
- **音量 (Volume) 曲线**：控制音频在不同速度区间的淡入淡出与响度变化。

## 4. 底部状态栏 (Status Bar)

底部面板主要用于提供实时的预览反馈，你可以在这里检验整体曲线的效果。

![底部状态栏](/docs/statusbar.png)

* **播放/停止 (Play / Stop)**：启动音频引擎，实时演算并播放当前所有未静音轨道的混合声音。
* **状态滑块 (Status slider)**：提供快捷按钮，用于在“牵引”、“惰行”和“制动”状态之间一键切换，以便测试列车在改变行驶状态时，声音过渡是否平滑自然。也可以通过快捷键`w`与`s`切换。
