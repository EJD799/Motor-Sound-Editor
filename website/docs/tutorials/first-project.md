# First Project

This tutorial walks you through a clean first project from creation to saved preview-ready state.

## Goal

By the end of this tutorial, you will have:

- created a new project
- assigned at least one audio file
- added a few keyframes
- previewed the project
- saved the result as `.msep`

## Step 1: Create the project

1. Open the Home screen.
2. Choose **New Project**.
3. Enter a project name.
4. Set a sensible maximum speed.
5. Set acceleration and braking defaults.
6. Enter the editor.

## Step 2: Confirm the active track

If the editor already has a usable default track, keep it. Otherwise:

1. Open the Track Layers panel.
2. Add a track.
3. Activate it.
4. Rename it to something descriptive.

## Step 3: Assign audio

1. In **Track Details**, open the file picker.
2. Choose a `wav` or `ogg` file.
3. Confirm the file name appears in the active track details.

## Step 4: Add initial curve shape

Use the chart editor or list editor to create a simple starter curve. You do not need a complex result yet. A practical starter pass is:

- one point near zero speed
- one point in the mid-speed range
- one point near the upper range

Do this for both pitch and volume if you want an immediately audible shape difference.

## Step 5: Preview

1. Set the simulator mode to **Traction**.
2. Set a current speed.
3. Press `Space`.
4. Listen for whether the track behaves roughly as expected.

If nothing is audible, check:

- whether the track has a file
- whether the track is muted
- whether the track is hidden

## Step 6: Save the project

1. Press `Ctrl+S` or use Save.
2. If needed, choose a location and file name.
3. Confirm the project is written as `.msep`.

## Result

You now have a real working project that can be reopened and refined.
