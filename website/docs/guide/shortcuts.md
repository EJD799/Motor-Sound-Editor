# Shortcuts

Motor Sound Editor is easiest to use when you learn the small set of shortcuts that support the main editing loop.

## Global editing shortcuts

- `Space`: play / pause preview
- `Ctrl+S`: save
- `Ctrl+Shift+S`: save as
- `Ctrl+Z`: undo
- `Ctrl+Y`: redo
- `Ctrl+Shift+Z`: redo
- `Ctrl+D`: clear the current selection
- `Delete`: delete the selected keyframe

## Interaction expectations

### Selection-sensitive editing

Several operations depend on which keyframe is selected. Before typing numeric values or pressing `Delete`, confirm that the correct point is active.

### Mode-sensitive editing

Chart behavior also depends on the active editor tool:

- Select
- Move
- Keyframe

If a click does not behave the way you expect, check the active tool first.

### Text fields and shortcuts

The app protects normal text-entry contexts so global editing shortcuts do not interfere when you are actively typing into an input field.

## Recommended daily shortcut set

If you only memorize a few commands, memorize these first:

- `Space`
- `Ctrl+S`
- `Ctrl+Z`
- `Delete`

These four cover the majority of everyday editing corrections.
