# Motor Sound Editor User Documentation

Motor Sound Editor is a dedicated desktop tool for building layered motor sound projects with visual curves, track-based audio assignment, real-time preview, and export packaging. This documentation is written for end users who want to create, edit, review, and export projects with confidence.

The docs are split into two parts:

- **Manual**: reference-style pages that explain the interface, core concepts, and day-to-day operations.
- **Tutorials**: guided walkthroughs that help you complete realistic tasks from start to finish.
